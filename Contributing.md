# Contributing to Awesome AI Tools

Thank you for contributing! Please follow these guidelines to keep the list high quality.

## Before Submitting

- Make sure the tool is **AI-powered** (not just a regular SaaS tool)
- Make sure the tool is **publicly accessible** (not internal or invite-only)
- Search existing entries to **avoid duplicates**
- The tool should be **actively maintained**

## Format

Every entry must follow this exact format:

```markdown
- [Tool Name](https://tool-url.com) - One sentence description, under 15 words.
```

**Rules:**
- Description starts with a capital letter, ends with a period
- Description is under 15 words
- No promotional language ("best", "amazing", "revolutionary")
- No trailing slashes in URLs
- Entries within each category are in **alphabetical order**

**Good example:**
```markdown
- [Upscayl](https://upscayl.org) - Free open-source AI image upscaler, no watermark, runs locally.
```

**Bad example:**
```markdown
- [Amazing Tool](https://tool.com) - The best and most revolutionary AI tool ever created!!
```

## How to Submit

1. Fork this repository
2. Create a new branch: `git checkout -b add-tool-name`
3. Add your tool to the correct category in alphabetical order
4. Commit: `git commit -m "Add Tool Name to Category"`
5. Push and open a Pull Request

## Pull Request Title Format

```
Add [Tool Name] to [Category Name]
```

Example: `Add Upscayl to Image & Design`

## Categories

Choose the most relevant category:

| Category | For tools that... |
|----------|------------------|
| Image & Design | Generate, edit, or upscale images |
| Writing & Content | Write, edit, or optimize text |
| Video & Animation | Generate or edit video |
| Audio & Music | Generate or edit audio/music |
| Code & Developer | Assist with coding tasks |
| Productivity | Automate or organize work |
| Marketing & SEO | Help with marketing and SEO |
| Data & Analytics | Analyze or visualize data |
| Education & Learning | Support learning and teaching |
| Business & Finance | Support business operations |
| Research & Search | Aid research and discovery |
| Other | Don't fit above categories |

## What Will Be Rejected

- Duplicate entries
- Tools that are not AI-powered
- Broken or inaccessible links
- Entries not following the format
- Promotional or biased descriptions
- Tools that are abandoned or no longer maintained
